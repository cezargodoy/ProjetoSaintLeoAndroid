# Rascunho do Projeto Android
Projeto para matéria de desenvolvimento de aplicativos móveis
Descrição do projeto

O Task Simple é um aplicativo Android simples para criação e organização de tarefas do dia a dia. O objetivo é permitir que o usuário registre atividades e acompanhe o que precisa ser feito de forma prática.

# Objetivo do projeto

Criar um aplicativo simples para aprender conceitos básicos de desenvolvimento Android, como interface, interação com usuário e organização de código.

# Problema que o aplicativo resolve

Muitas pessoas esquecem tarefas ou acabam se organizando em papel ou anotações soltas.

# O aplicativo resolve isso permitindo:

registrar tarefas
visualizar tarefas pendentes
marcar tarefas como concluídas
